# 規則適用範圍與優先順序

## 目的

規則是常駐的，語言專屬規則必須有明確範圍以避免衝突。

## 優先順序

1. **語言規則優先於核心規則**（若有衝突）。
2. **核心規則適用所有語言**，除非被語言規則覆蓋。

## 如何判斷語言範圍

可使用以下訊號：
- **Java**：存在 `pom.xml` 或修改 `.java` 檔案
- **TypeScript/JavaScript**：存在 `package.json` 或修改 `.ts/.tsx/.js/.jsx` 檔案
- **Go**：存在 `go.mod` 或修改 `.go` 檔案

若不確定，優先以修改的檔案類型判定。
