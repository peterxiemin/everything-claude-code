# 重构清理 (Refactor Clean)

通过测试验证，安全地识别并移除冗余代码：

1. 运行冗余代码分析工具：
   - **knip**：查找未使用的导出和文件
   - **depcheck**：查找未使用的依赖项
   - **ts-prune**：查找未使用的 TypeScript 导出项

2. 在 `.reports/dead-code-analysis.md` 中生成详细报告

3. 按风险程度对发现的问题进行分类：
   - **安全 (SAFE)**：测试文件、未使用的工具类
   - **谨慎 (CAUTION)**：API 路由、组件
   - **危险 (DANGER)**：配置文件、主入口点

4. 仅建议进行安全的删除操作

5. 在每次删除前：
   - 运行完整的测试套件
   - 验证测试通过
   - 应用更改
   - 重新运行测试
   - 如果测试失败，则回滚更改

6. 显示已清理项的摘要

严禁在未运行测试的情况下删除代码！
