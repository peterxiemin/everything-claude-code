# 构建与修复 (Build and Fix)

增量修复 TypeScript 和构建错误：

1. 运行构建：`npm run build` 或 `pnpm build`

2. 解析错误输出：
   - 按文件分组
   - 按严重程度排序

3. 针对每个错误：
   - 显示错误上下文 (前后各 5 行)
   - 解释问题原因
   - 提出修复方案
   - 应用修复
   - 重新运行构建
   - 验证错误已解决

4. 在以下情况停止：
   - 修复引入了新错误
   - 尝试 3 次后相同错误依然存在
   - 用户请求暂停

5. 显示总结：
   - 已修复的错误数
   - 剩余的错误数
   - 新引入的错误数

为了安全起见，每次仅修复一个错误！
